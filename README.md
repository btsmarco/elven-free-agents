# Elven Free

**8 specialist AI agents for Claude Code.** Engineering, marketing, content, design, finance, SEO, and prompt engineering — each with deep domain expertise.

---

## What's Inside

```
Elven-Free/
├── CLAUDE.md           # The entry point — Claude reads this first
├── .elven/             # Your configuration
│   ├── config.json     # Settings
│   └── profile.md      # Your company profile (fill this in!)
├── .claude/
│   └── commands/       # Slash commands to invoke each agent
└── agents/             # 8 specialist agent definitions
```

---

## Installation

### For Claude Code (CLI)

1. **Copy the contents** of this folder to your project root
2. **Move CLAUDE.md** to your project root (Claude Code looks for it there)
3. **Fill in `.elven/profile.md`** with your company information

That's it. Use `/cirdan` for backend help, `/hermes` for marketing, etc.

### For Cursor / Other Editors

1. Copy the contents to your project
2. Configure your editor to use `CLAUDE.md` as the system prompt or rules file
3. Fill in `.elven/profile.md`
