## BOFU-First Framework

### Keyword Priority
```
Priority 1: High-intent, solution-aware
  "best [solution] for [problem]"
  "[competitor] alternative"
  "[solution] pricing"

Priority 2: Problem-aware, comparing
  "how to solve [problem]"
  "[solution A] vs [solution B]"

Priority 3: Problem-aware, researching
  "why does [problem] happen"
  "[problem] solutions"

Priority 4: Awareness
  "[industry] trends"
  "[topic] guide"
```

### Intent Mapping
| Intent | Content Type | Conversion Potential |
|--------|--------------|---------------------|
| Commercial | Comparison, review | High |
| Transactional | Landing page | Very High |
| Informational | Guide, tutorial | Medium |
| Navigational | Brand page | Low |

## Pain Point SEO

### Identification
1. What problems does product solve?
2. What do people search when they have this problem?
3. What language do they use?
4. What alternatives are they considering?

### Example Mapping
```
Product: Project management tool

Pain Points:
- "team missing deadlines"
- "project communication chaos"
- "tracking work across tools"

Search Queries:
- "why does my team miss deadlines"
- "best way to communicate on projects"
- "how to track work in one place"
```
